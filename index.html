Frequently asked questions

What is self-employment tax?

It generally funds Social Security and Medicare for net earnings from self-employment. It is distinct from federal income tax.

How much should a 1099 worker save for taxes?

There is no universal percentage. Income, expenses, household facts, location, credits, and withholding all matter. Use a range and current professional guidance.

Does this site file taxes?

No. It provides educational browser-based estimates only.

Are state taxes included?

Only when you manually enter a state planning rate in tools that support it. No state rates are supplied.
